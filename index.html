Trail Route Editor

Design and optimize trail routes using real terrain data

Getting Started

1
Load a DEM — Click Load DEM and select a GeoTIFF elevation raster. The app generates a hillshade, contour lines, and 3D terrain from this file. Cropped DEMs covering just your project area work best.
2
Load or draw trails — Click Load Trails to import existing routes (GeoJSON, KML, or GeoPackage with local server), or click New Route to draw one from scratch by clicking on the map.
3
Edit vertices — Select a trail from the dropdown. Drag vertices to reshape. Right-click a vertex to delete it. Click a segment midpoint to insert a new vertex.
4
Optimize — Click Optimize Route to run the spring-mass optimizer. It reshapes the trail to meet a target grade while following the terrain.
5
Export — Download your edited trails as GeoJSON or KML.

App Layout

Top toolbar — Trail selector, basemap toggle, load/export/draw/optimize buttons
Map — Interactive map with hillshade, contours, and 3D terrain. Right-click drag to tilt.
Bottom panel — Elevation profile, slope profile, and trail stats. Drag the top edge to resize. Click the charts to zoom the map to that location.

Tips

🏷
Trail naming matters — Trails loaded with a Name or name property get listed individually in the selector. Trails without names are auto-assigned names like “Trail 1”.
❄
Freeze vertices before optimizing — Shift+click to freeze/unfreeze individual vertices (turns blue). Ctrl+Shift+click two vertices to freeze the range between them. Use “Freeze Flat” in the optimizer to auto-freeze segments already below 5% grade. Frozen vertices stay fixed during optimization.
📋
Optimizer creates a copy — Running the optimizer adds a new trail called “[name] - Optimized” so you can compare before and after side by side.
⚠
Target Grade vs Max Segment Grade — Target Grade sets the overall average. Max Segment Grade is a hard cap on any individual segment. Set target to 7% and max to 15% to allow some steepness while keeping the average reasonable.
🎯
Recenter if the map gets stuck — If zooming into 3D terrain makes the map unresponsive, click “Recenter” to reset the view.
⚡
Contours are cached — First generation takes a few seconds. Subsequent loads of the same DEM are instant.
⌨
Keyboard shortcuts — Ctrl+Z undo vertex moves. While drawing: Ctrl+Z removes last point, Enter finishes, Esc cancels.
📡
Works offline — Once loaded, everything runs in your browser. No data is sent to any server. The local Python server is only needed for GeoPackage files and caching.
